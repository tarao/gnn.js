Class Detail
GNN.Hash()
Creates a hash table.

Methods and property gets in GNN.Hash.prototype are also available as static methods in GNN.Hash with taking the first argument as an array. In this form, an object can be used as a hash.

new GNN.Hash({a:1, b:2, c:3});
// => Hash {a: 1, b:2, c:3}
new GNN.Hash(new GNN.AssocArray(['a', 1], ['b', 2], ['c': 3]));
// => Hash {a: 1, b:2, c:3}
new GNN.Hash('a', 1, 'b', 2, 'c', 3);
// => Hash {a: 1, b:2, c:3}
Parameters:
{GNN.AssocArray|object|...*} Optional
If the odd number of arguments are specified, then the first argument must be a GNN.AssocArray or an object. The keys and values of the argument are copied to the hash table. Otherwise, the arguments must be an array. The ith element of the array is a key and the (i+1)th element of the array is the value associated with the ith key for every even number i.

{GNN.Hash} map(fun, thisp)
Returns a new hash table with the results of calling the given function on each value of the key-value pairs. It stores fun(k, this[k], this) to the new hash table at key k for each key k in this. 
GNN.Hash({ a:1, b:2, c:3, d:4 }).map(function(k,v){return v*v;});
// => { a:1, b:4, c:9, d:16 }
GNN.Hash({ a:1, b:2, c:3, d:4 }).map(function(k,v,o){o[k]=0;return v+k;});
