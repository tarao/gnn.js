Class Detail
<inner> GNN.Tester-TestHandler(parent)
Creates a test client. The client is set up by the test server in a test environment. Inside the environment, the instance of the client is accessible as GNN.Tester or simply Tester. The client is used in each test environment to run tests and reports the result to the test server. 
// in test_something.js

Tester.run(function(t) {
    t.ok(true);
}, 'test case 1');

Tester.run(function(t) {
    t.error('fail!');
}, 'test case 2');
